Setting the Detector Voltage

The first time you run a scan with SpecsLab Prodigy, it is necessary to set the detector voltage. By default, this is set to 0 V. The correct operating voltage is shown in the test report for your analyzer.

 

NOTE:
As the detector ages, you will need to increase the operating voltage. You can find the optimum operating voltage with a Detector Voltage Scan and enter it into the software as described below.

 

To set the detector voltage:

1. Select the Device Controls view. This is usually a tab that shares space with the Experiment Editor. If it is not open, select Views/ Device Controls from the menu bar.
2. Click Connect to establish a connection with the analyzer.
3. Click the icon to expand the view.
4. Select the Calib. Voltages tab.

 

 

5. Enter the detector voltages from the test report in the fields provided.
6. Click Store. The values will be saved to the registry and will be used each time you connect to the detector.

 

You can click Disconnect to break the connection to the analyzer. If you want to measure a spectrum now, you can leave it connected.
