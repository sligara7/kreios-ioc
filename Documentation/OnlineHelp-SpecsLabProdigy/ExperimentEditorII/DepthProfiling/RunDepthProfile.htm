Running the Depth Profile

Defining spectra for the depth profile experiment is the same procedure as for a normal scan.

 

To run the depth profile:

1. Click the icon to add a spectrum group to the depth profile.
2. Define the spectrum group in the normal way.
3. Click to validate the scan and start data acquisition.

 

Data is shown in the Plot View. For a better view of the scans, you can click to display the Z axis, then select Etch Time for the Z axis. You can also view the data in the Image View. For more information, please refer to:

■    Changing the Axis Labels.

■    Displaying Spectrum Components.

■    Opening Data in the Image View.
