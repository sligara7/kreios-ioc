Ramping

A ramp is a linear change of a parameter value over a specified time period. Ramps run while a spectrum group is being acquired.

 

NOTE:
You need to add a device to the experiment that offers the ramping parameter.

 

When you define a ramp, it is possible to add another parameter with a conditional relationship to pause the ramping. This is useful, for example, for stopping the ramp if the pressure rises above a certain level.

 

Although it is not possible to simultaneously ramp a parameter and record it with spectrum data, there are methods for monitoring the ramping parameter during the ramp.
