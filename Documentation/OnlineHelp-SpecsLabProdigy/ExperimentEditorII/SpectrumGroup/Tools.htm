Tools

There are a set of icons below the list of spectrum definitions which add, remove or perform actions on spectrum definitions. The table below lists function of each icon.

 

You can select multiple spectra as follows:

■    Hold SHIFT and click a spectrum to select a range.

■    Hold CTRL and click a spectrum to add it to the selection.

 

Icon Description
Adds a spectrum definition to the group. The values in the spectrum are based on the currently selected spectrum or, if no spectrum is selected, on the last spectrum in the table.

Deletes the selected spectra.
Shows the periodic table, if available. Selecting an element and excitation in the table adds a suitable energy range to the spectrum definition.
Validates the selected spectra.

Uses the electron kinetic energy to define the spectrum.
Sets the energy settings for the spectrum definition to binding energy, as calculated from the source energy.
Unlocks selected spectra, allowing you to modify the definition after measurement. This will delete the data in the spectra.

Displays a menu with the same entries as the context menu.
