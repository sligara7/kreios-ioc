Heating the Sample

Operating the EBH 150 Device Control is analogous to front panel operation.The example below shows how to use the software for heating the sample with emission regulation. You can adapt the procedure for other operation modes.

 

NOTE:
You can only perform temperature regulation with the first temperature sensor. In addition, only temperature channels that have been assigned a sensor are displayed in this view. A green indicator means the sensor is working. Red means a fault, e.g. the sensor is not connected.

 

To heat the sample with emission regulation:

1. Select Emission from the Mode drop-down list. The Emission entry field will become active.
2. Set the desired emission current, e.g. 10 mA.
3. Select On from the HV drop-down list.
4. Set a value in the Voltage field, e.g. 800 V.

 

 

5. Click Apply. Your changes will be sent to the EBH 150 and it will ramp up the filament current until the desired emission current is achieved. It will then regulate at this current.

 

To stop the sample heating, click one of the following buttons:

■     Filament Standby. The high voltage will be switched off and the filament current reduced to its standby value. This is the recommended mode if sample heating is to be performed frequently, since it is faster to ramp the filament up to its operating value from the standby value.

■    Filament Off. The filament and high voltage are switched off.
