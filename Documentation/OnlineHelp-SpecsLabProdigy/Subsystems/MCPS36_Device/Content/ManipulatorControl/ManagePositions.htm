Adding Positions to a List

To add a new point to the current list:

1. Click the Add Position button. A new line will appear in the position list, showing the name and the settings of all motors.

 

 

2. Double-click the name in the Position set column. Enter a name for the position.
3. To add further positions to the list, move the manipulator to a new position and click Add Position.

 

Retrieving Positions from a List

To move the motor to a position in the list:

4. Select the desired position in the list. A radio button indicates the selected position.
5. Click Move To Position. The motor settings in the list will be transferred to the Destination fields in the table in the top pane. At the same time, the motor will start to move to the destination.

 

 

Editing Lists

There are a number of options for editing lists. The table below summarizes your options.

 

NOTE:
For most of these actions, you need to first select the item in the list. A radio button shows the currently selected list item.

 

Action Description
Double-click fields in table The selected field becomes active; you can then enter a new name or axis setting.
Click Overwrite Position Similar to Add Position. The current motor positions overwrite those in the selected list item.
Click Delete Removes the selected position from the list. A dialog will appear asking you to confirm the delete.
Click Move Entry Up These buttons move the selected position up and down in the list. This is useful if you want to export a list of manipulator positions—you can set the positions a suitable order.

Saving and Loading Lists

You can save position lists by clicking the and load them at a later date from the drop-down list.

 

NOTE:
The default location for configuration files is settings/ [Device Name] in the SpecsLab2 installation directory. The drop-down list shows all configuration files in this directory. It also allows you to select previously loaded files, if these were stored in different locations; these are removed from the list after restarting the program.

 

Exporting Lists

You can export lists as a text file. In this way, is useful for preparing position lists.

 

Exported lists have the following structure:

■    A header describing the file. All comment lines start with a #.

■    The name of the point is included on its own line as a comment.

■    Following is a line containing the points for each axis setting. The values are comma-separated.

 

The following example shows a file containing three position settings for a 4-axis manipulator.

 

#SPECS position list file

# Pos1

2.50,2.00,55.96,20.00

# Pos2

2.40,2.00,55.96,20.00

# Pos3

2.30,2.00,55.96,20.00

 

To export a list:

6. Click Export. A file browser will open.
7. Select a location and filename and click Save. The file is saved without an extension.
