PHOIBOS

The PHOIBOS Device Control allows you to manually set voltages in the analyzer and detector. It is primarily used for setting up and calibrating the instrument. The following topics are covered:

■    Setting operating parameters and switching on the analyzer.

■    More advanced settings for the detector and for calibration.

■    Viewing analyzer voltages.

 

 

Before using the PHOIBOS Device Control, you need to make a connection:

■    Click Connect. There will be a pause while the connection is made. On establishing the connection, the Device Control frame turns blue. If you have configured the device as part of an experiment in the Experiment Editor, the Device Control is already connected to the analyzer.

 

NOTE:
The device can also be included in experiments.

 

Clicking the Disconnect button breaks contact with the instrument. All controls are inactive. There are two buttons, and , which collapse and expand the Device Control respectively.
