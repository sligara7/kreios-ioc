Settings

The Settings tab contains some operational parameters that

Parameter Description
Maximum Count Rate [kcps] Sets a maximum count rate to protect the detector. If a higher count rate is detected, the detector is switched off.
Analyzer Standby Delay [s] After completing the final scan, the detector voltages are switched off; all other analyzer voltages stay at their final scan values until this delay elapses. If you are performing a series of scans, you can change this delay so that the analyzer voltages are not put into standby between scans.
Skip Delay Up/Down When starting a scan, all analyzer and detector voltages are switched on. There is (by default) a short delay to allow the voltages to stabilize. You can disable the delay by checking this box.

 

After setting the parameter values, click Apply and Save to accept your changes and store the values for future sessions.
