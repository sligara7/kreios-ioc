Calibration Voltages

The Calibration Voltages tab contains settings for the detector. The features are listed in the table below.

 

Parameter Description
Screen Voltage 2D detectors only. Please refer to the test report to find a suitable screen voltage for your detector.
Bias Voltage Electrons The detection efficiency of the detector is proportional to the bias applied to the detector. The efficiency is different for electrons and ions. Typical values are preset: +90 V for electrons; −2000 V for ions.
Bias Voltage Ions
Detector Voltage The detector voltage. This is given in the test report for your detector. Aging of the detector will cause this operating value to increase—you should run a detector voltage scan every ~2 months to find the correct operating conditions.
Ramp Time Sets a time during which the detector and/ or screen voltage is ramped linearly from zero to the set value. This is normally used for conditioning a detector. Please see the detector manual for this procedure.

 

After setting the voltages in the tab, you can send them to the power supply:

■    Apply—sends the setting for this session.

■    Store—saves the setting in the registry.
