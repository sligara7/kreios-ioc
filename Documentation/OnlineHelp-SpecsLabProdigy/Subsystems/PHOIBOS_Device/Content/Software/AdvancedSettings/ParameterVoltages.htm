Parameter Voltages

The Para. Voltages tab contains additional settings for controlling the analyzer. These are used for setting up, tuning and modifying analyzer performance. The parameters listed depend on the type of analyzer.

 

In general, you should not change these settings without instructions from SPECS support.

 

NOTE:
The parameters listed in this view reflect the content of the HSA 3500plus configuration file.

 

There are two buttons at the bottom of the tab:

■    Apply—sends the setting for this session.

■    Store—saves the setting in the registry.
