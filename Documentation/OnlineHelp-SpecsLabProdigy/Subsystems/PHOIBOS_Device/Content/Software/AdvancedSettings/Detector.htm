Detector

The detector tab is only shown for 2D detectors. The sections below describe the features in each section of the tab.

 

Angular Mapping

Check the Use Image Correction box for the necessary geometrical transformation to the image, compensating effects due to electron optics. The correction matrix is stored in a configuration file.

 

You can specify the number of channels with the two Channels fields:

Energy Channels × Non-Energy Channels (e.g. angular dispersion direction)

 

A higher number of channels offers increased resolution; however, the number of counts in each channel will decrease, which can lead to a noisy signal.

 

See Active Area and Channels for more information.

Detector Active Area

The detector active area is the area that is used for data acquisition. In general, you should select the largest possible area on the image—remember however that the camera image is circular, so you should aim for the largest square within this circle, so that all channels have the same contribution to the signal.

 

Energy channels and non-energy channels have a specified width in pixels. You need to be able to fit a whole number of channels into the picture. In order to make best use of the selected area, observe the following points:

■    The specified width should be exactly divisible by the number of energy channels.

■    The specified height should be exactly divisible by the number of non-energy channels.

 

If these conditions are not fulfilled, parts of the border of the image will not be read. A warning message will appear in this case before the values are accepted.

 

The values are preset for the optimum area, so you do not normally have to change them. The procedure below

describes the steps required to obtain and enter new settings, should you wish to change the defaults.
