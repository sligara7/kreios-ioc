Standard Channel

The Std. Channel tab is only available for MCD detectors. It provides gain and offset factors for each channel in the PCU-300 preamplifier and counting unit. This can be used to correct differences between channels and is sometimes useful for correcting a noisy CEM.

 

Please refer to the SCD/ MCD-5/ MCD-9 detector manual for details about the PCU-300 and its adjustment.
