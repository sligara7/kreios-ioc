Viewing Ion Source Operating Parameters

Clicking the icon until the Device Control is fully expanded reveals the information panel, which shows all operating and setup parameters. These are described fully in the PU-IQE 12/38 manual. The table below has a brief overview of these parameters. All parameters are read-only in this section.

Blanking X/Y Settings to trigger a TTL signal that indicates if the beam is near the limits of the scan.
Phi Correction factors for the sample angle relative to the source.
Theta
Blank Level Indicated the polarity of the TTL signal produced in conjunction with the blanking function.
Distance L Distance between sample and source.

Deflect Y
