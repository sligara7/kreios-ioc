Changing the Scan Settings

When you click the icon once in the IQE Device Control, additional parameters appear. These allow you to change the area covered by the ion beam as well as the emission current.

 

The ion beam can be scanned over an area up to 10 × 10 mm, with a step size of 0.1 mm. The parameters Pos X, Pos Y and Time Per Dot also influence the scan size and speed. See Editing Configuration Settings as well as the PU-IQE 12/38 manual for more information.

 

The filament current in the source cannot be set directly—it depends on the (regulated) emission current set by the user.

 

To change scan settings:

1. Select a setting for the source from the drop-down list.
2. Check a box next to the parameter that you wish to edit. The field will become active.
3. Enter a new value in the field.

 

Your settings will be used when you start the source, either in sputter or manual mode. The right columns next to the parameters show the source status.
