Editing Configuration Settings

The PU IQE 12/38 has a number of preset configurations. These contain parameter sets that describe a specific mode of operation. There are ten presets available. You can edit these presets to customize the source operation, then save your values for future use.

 

NOTE:
You can change settings and view their effects on performance on-the-fly. Saving your changes is optional.

 

To edit an ion source configuration:

1. Select a setting from the drop-down list. The settings will be loaded and displayed in the fields.
2. Check boxes next to the parameters you want to edit, and change their values accordingly. The parameters are described fully in the IQE 12/38 operating manual; the table below provides a summary.
3. When you are ready to save your configuration, select a setting in the Store Ion Configuration drop-down list and click Overwrite. Your new settings will be stored in the IQE 12/38 under the selected setting. Note that the old values in this setting will be lost.

 

 

Parameter Description
Width X Defines the width of the area scanned by the beam. The maximum width in each dimension is 10 mm. Note that this is affected by the Pos X and Pos Y settings.
Width Y
Emission Controls electron emission current from the filament. The filament current cannot be controlled directly.
Pos X Sets the center point of the scanned beam. The maximum is ±5 mm in each dimension. This affects the maximum area that can be scanned (see Width X and Width Y). If the beam center is not at 0,0, the full 10 × 10 mm scanning area is not possible.
Pos Y
Energy Ion beam energy.
Extractor Parameter to extract ions from the ionization region in the source. This is a percentage of the Energy value. This setting, and the extractor current obtained (see Viewing Ion Source Operating Parameters) is dependent on the gas type and pressure. Please see the user manual for more details.
Focus 1 Set the values of the focus electrodes. These are a percentage of the Energy value. More information is available in the source manual; note that broad beam FWHM ~800 μm means F2 = 0, fine beam mode ~120 μm F1 = F2~77 %.
Focus 2
Time Per Dot Scanning rasters the beam spot in 100 μm intervals. Time Per Dot defines the dwell time for each point. This setting thereby influences the scanning frequency for an area (see Width X and Width Y).
Scan Mode

Select an item from the drop-down list:

■   Internal with Mon—The X Mon and Y Mon connectors on the power supply act as outputs for the scan signal, allowing the scan signal to be monitored by other equipment.

■   Internal without Mon—The equipment will scan in X and Y directions, but the monitor signal on the rear panel is not active.

■   External—The X Mon and Y Mon connectors on the power supply act as inputs, allowing you to control the scan from other signals.
