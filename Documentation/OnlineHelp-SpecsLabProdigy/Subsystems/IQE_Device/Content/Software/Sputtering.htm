Controlling Sputtering

The basic settings of the IQE Device Control allow you to switch the source on and off with pre-defined settings. The ion source is often used in conjunction with a VCU 1000, which can set and regulate chamber pressure.

 

There are two tabs in the main section of the Device Control for sputter operation and manual operation.

 

Sputter Operation

The Sputter Operation tab allows you to switch the ion source on for a specified period of time. The source will operate according to the selected stored presets.

 

To use sputter operation:

1. Set the desired gas pressure in the chamber, either setting this with the VCU 1000 Device Control or by operating the leak valve manually on the chamber.
2. Select a setting from the drop-down list. Please see the PU-IQE manual for a full description of settings.
3. Enter a time for the sputter exposure.
4.

Select a mode for the ion gun after the sputter exposure is finished:

Standby—Filament current is reduced to the standby limit, HV off.

Emission On—Filament remains at operating value, HV off.
5. Click Start to switch on the ion source for the specified time.

 

The information field shows the number of times the ion source will scan over the area. This is determined by the Width and Time Per Dot parameters—see Editing Configuration Settings

 

 

Manual Operation

The Manual Operation tab allows you to switch the source on and off with a selected setting.

 

To operate the source in manual mode:

6. Set the desired gas pressure in the chamber, either setting this with the VCU 1000 Device Control or by operating the leak valve manually on the chamber.
7. Select a setting from the drop-down list.
8. Select Emission On from the Operating Mode drop-down list.
9. Select On from the High Voltage drop-down list.
10. Click Apply to switch on the source.

 

To switch off, you can either select Off from the drop-down list, or Standby to switch off the high voltage and reduce the filament down to the pre-defined standby level. Always press Apply to accept your changes.

 

There is also a degas setting which you can use to ramp up the filament to its normal operating current after bakeout or in order to condition a new filament.
