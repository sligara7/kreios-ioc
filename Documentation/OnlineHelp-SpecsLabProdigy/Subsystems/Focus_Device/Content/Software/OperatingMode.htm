Source Operation

The initial view of the source provides basic controls for setting the operating mode of the source.

 

To switch the source on:

1. Select Operate from the Operating Mode menu. See the table below for an explanation of the other available menu items.
2. Select an anode from the Anode menu.
3. Click Apply. The source will enter operating mode.

 

 

To switch the source off:

■    Select Off from the Operating Mode menu, then click Apply.
