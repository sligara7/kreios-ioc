Overview of all Features

Clicking the icon twice expands the Device Control to its maximum extent. The table below lists all the features in the device.

 

 

Feature Description
Operating Mode See Source Operation for a description of all operating modes.
Anode Selects which anode is used for generating X-rays. The type depends on your source.
Power Enter the desired power output from the source. This is limited by the setting in the power supply. The limit depends on the anode type.
Voltage The anode voltage in the range 0â€¦15 kV.
Emission The emission current in the source.
Temperature The internal temperature of the power supply.
Filament The filament current.
Emission + Leakage The total anode current. This consists of the filament emission current and the leakage (i.e. water) current.
Disconnect Breaks the connection between the computer and power supply. The instrument can then be controlled from the front panel.
Apply Sends your settings to the power supply.
