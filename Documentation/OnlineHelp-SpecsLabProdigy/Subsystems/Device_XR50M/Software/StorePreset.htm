Storing Presets

The XR 50 M is switched on using presetsâ€”operating values for certain operating modes and parameters. You can change settings and reuse them in future by saving them in a preset.

 

To store a preset:

1. Select a preset.
2. Adjust the operating parameters as necessary.
3. Expand the device control by clicking the icon once.
4. Click Store Preset. A dialog will appear.
5. Select a preset from the drop-down list.
6. If you want to rename the preset, check the option box and enter a new name in the field provided.
7. Click OK to store the preset.

 

NOTE:
You cannot overwrite the default presets.
