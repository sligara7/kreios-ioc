Controlling the Heater Pockets

You can click the icon in the initial view to expand the device control. This reveals three tabs that contain different methods for controlling the heater. Expand the sections below for more information about these tabs.

 

Caution!

When heating, remember that the pyrometer does not report temperatures below 750 °C. The output for all temperatures below this value is 0 °C.

 

Caution!

The pyrometer can only read the temperature of one pocket at a time. When regulating a pocket temperature, always make sure that the cathode cartridge is in the correct orientation so that the pyrometer can monitor the pocket.

Failure to set the correct position can lead to damage to the cathode.

 

Temperature ramping

The temperature ramping tab allows you to set the temperature of an individual pocket and hold it for a specified time:

1. Select a pocket from the drop-down list.
2. Enter a target temperature.
3. Enter a time for the temperature to be maintained in the Hold Temperature field.
4. Set the Timer Threshold to a suitable value. This is the tolerance of the target temperature at which the timer starts operating.
5. Click Start. You will see in the main status area that the heater current for the pocket is ramped until the target temperature is reached. The timer will then count down. After the timer reaches zero, the current will be switched off.

 

There are two other buttons to control the ramping:

■    Hold Current maintains the current at its present value. This is useful if the cathode has reached a stable regulated value. You can hold the current for the pocket constant and then start the regulation for another pocket.

■    Switch Pocket Off sets the current for the pocket to zero.

 

 

Current ramping

The current ramping tab allows you to ramp up the current to any combination of pockets:

1. Check the option boxes of the pockets you want to heat.
2. Enter the target current.
3. Enter the rate of the current rise in mA/s in the Current Step field.
4. Click Start. You will see in the main status area that the heater current for the pocket is ramped until the target current is reached.

 

You can stop the heating by clicking the Switch Pockets Off button in the main window.

 

 

Check pockets

Clicking the Check button in this tab sends a low current through all pocket circuits too see if the pockets are operational. A green indicator shows that the pocket is connected and working.

 

This feature is only intended for test purposes. If the pockets are heating when you press Check, the following occurs:

■    The heating current to all pockets is immediately reduced to near zero in order to perform the check.

■    When you resume heating, the heaters return to their previous currents, without ramping.
