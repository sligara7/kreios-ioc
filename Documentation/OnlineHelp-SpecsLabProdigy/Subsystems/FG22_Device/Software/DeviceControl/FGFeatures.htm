Overview of all Features

Clicking the icon once shows the Filament Limit setting.

 

On clicking the icon again, the Device Control expands to its maximum extent. The table below lists all the features available.

 

 

Feature Description
Emission Sets and reads the emission current.
Energy Sets and reads the beam energy. The maximum value depends on the range selected in Energy Mode.
Filament Limit Sets and reads the filament limit current.
Extractor Reads the extractor value.
Temperature Reads the internal temperature of the power supply.
