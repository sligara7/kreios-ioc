Changing the Settings

Clicking the icon expands the device control to reveal the settings. These settings are summarized in the table below.

 

Item Description
Check box Check the boxes to activate the respective probe.
Probe The name of the probe. You can edit this setting. The name is used in the status area of the device as well as in the Experiment Editor.
Bias

Sets the bias. The following options are available:

■   None—no bias is used.

■   +40 V—switches on the internal +40 V bias.

■   –40 V—switches on the internal –40 V bias.

■   External—allows you to apply an external bias to the signal.
Range

Select a range for the current reading:

■   0…2 nA

■   0…0.2 μA

■   0…20 μA

■   0…2000 μA

■   Autorange

 

After making changes to the settings, click Store to apply them and save them to the registry.

 

Clicking the Read button restores the current registry settings.
