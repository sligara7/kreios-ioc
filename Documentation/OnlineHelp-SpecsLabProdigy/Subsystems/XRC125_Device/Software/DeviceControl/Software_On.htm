Switching On the Source

In its initial state, with all levels collapsed, the XRC-125 MF Device Control allows you to switch the source on and off using presets.

 

To operate the source:

1. Select an item from the Preset drop-down list.
2. Click Activate Preset. The source will start in the selected configuration. The status is shown in the field.

 

 

To switch off the source, you can click one of the following buttons:

■    Sleep—switches off the filament and the high voltage. This is the same status as when the instrument is initially switched on.

■    Standby—turns the filament down to the standby level and switches off the high voltage. This is better if the source is being switched on and off, as the filament ramps up to its operating current more quickly.
