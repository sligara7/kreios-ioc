Anode Positions Tab

The Anode Positions tab shows the preset anode positions and allows you to change the location of each anode position. There are ten anode positions stored in the XRC-125 MF, together with additional positions used for calibration and characterization of the source.

 

Running an overview scan

The overview scan scans over the features of the anode assembly, including the aluminium anode, the silver edge, molybdenum edge and Faraday cup. It measures electron intensity (from the secondary electron detector) as a function of distance.

 

Overview scans always use preset 1. As set at the factory, this is a low power setting that will cause minimum wear to the anode.

 

To perform an overview scan:

■    Click Start Overview Scan. The electron intensity as a function of anode position is shown in the plot.

 

 

 

NOTE:
The XRC-125 MF manual contains details of the low level commands used to perform this scan as part of the software interface description.

 

Setting new positions

After running an overview scan, you can change the defined positions as necessary:

1. Click the radio button next to the position you want to change.
2. Zoom into the area in the overview scan that contains the position you want to select. You can either use the icon or select an area in the plot.
3. Select the cursor in the toolbar.
4. Click the new position in the overview scan. The label and the setting is updated.
5. Click in the toolbar. This prevents you from accidentally changing the setting by clicking somewhere in the overview scan.

 

 

NOTE:
You can undo your changes and restore the positions from the XRC-125 MF by clicking Load Positions.

 

6. Set other positions as desired, then click Store Positions. This will save all of your changes to the XRC-125 MF, overwriting the existing settings. You need to confirm this action.

 

GUI features

The Calibrate Anode Positions tab is divided into three regions:

■    Positions. These are the various positions on the anode assembly. You can select one of these to set its position.

■    Toolbar and control buttons. These are listed in the table below.

■    Graphical view. This shows a representation of the anode profile. All positions are shown.

 

Feature Description
Load Positions Loads the anode positions from the XRC-125 MF. This overwrites the settings in the view.
Store Positions Saves the current settings in the view to the XRC-125 MF. You will be asked to confirm this action before the settings in the XRC-125 MF are overwritten.
Overview Scan Starts the overview scan, as described above.
Ref Drive Calibrate the motor position—sends the motor to the start position and zeros this position. All subsequent positions are calculated (in steps, with a conversion factor) from this position.
