Setting Preset Values

On clicking the icon twice from the initial view, the Device Control offers a full set of instrument parameters, including limit settings. These settings allow you to modify preset configurations, then save the new preset values to the XRC-125 MF.

 

There are also a set of tabs for other functions, which will be covered in the sections below.

 

Caution!

When you save a preset, the original values in the XRC-125 MF are overwritten and cannot be retrieved.

 

To change preset values:

1. Select a preset from the Preset drop-down list.
2. Check the parameter that you want to adjust. It will be become active.

 

 

3. Enter a new parameter value.
4. Click Store. The Store Preset dialog will open.

 

 

5. Select a preset from the drop-down list. You will overwrite this preset with your new values.
6. If you want to give the preset a new name, check the Replace Preset Name box and enter a new name.
7. Click OK. The new values (and name) are written to the selected preset in the XRC-125 MF.
