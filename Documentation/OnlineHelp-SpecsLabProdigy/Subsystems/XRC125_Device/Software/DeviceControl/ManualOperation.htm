Manual Operation and Anode Position

By clicking the icon once in the initial view, you see additional controls for the following:

■    Manually setting the source operating parameters.

■    Moving the anode to a stored position.

 

Manual operation

All operating parameters for the electron gun are available. For setting limits, please also see Setting Preset Values.

 

Please also refer to the μ-FOCUS 350 user manual for information about the electron gun electrodes and advice about adjustment.

 

Caution!

Adjusting operating parameters can damage the anode. Please refer to the μ-FOCUS 350 user manual before changing operating parameters.

 

To change operating parameters:

1. Select a preset from the Preset drop-down list.
2. Click Activate Preset to start the source with the selected parameters. You will see that the current operating parameters are updated.
3. Check the parameter that you want to adjust. It will be become active.
4. Enter a new parameter value.
5. Click Activate Preset to accept the value.

 

 

Setting the anode position

To move the anode:

6. Select a position in the Anode drop-down list.You can view the positions by pressing the icon to expand the view and selecting the Anode Positions tab.

 

 

NOTE:
The positions for the silver edge, Faraday cup, etc are not available in this list. These positions are only accessed as part of the calibration scans. Leaving the source switched on in these positions will damage the anode assembly.

 

7. Click Move. The anode will be moved to the selected position. The field next to the Move button will be updated when the action is completed.
