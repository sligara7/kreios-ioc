Anode Scan Tab

The anode scan moves the anode over a specified area (usually the silver edge) and records the current from the secondary electron detector. A line is then fit to the results, showing the width of the electron beam. You can therefore calibrate the beam width for different presets, i.e. different electron gun settings.

 

To perform an anode scan:

1. Select a preset that you want to use for the scan.
2.

Select a location for the anode scan from the Scan Range drop-down list:

■    Width at Ag Edge is the standard option using the silver edge on the anode assembly.
When selecting this option, you can adjust the Width parameter to change the scan range—the Ag edge is at the center of the range.

■    Width at Mo Edge uses the molybdenum edge on the anode assembly. When selecting this option, you can adjust the Width parameter to change the scan range—the Mo edge is at the center of the range.

■    Start/ End Position allows you to define your own range for the scan. The Start and End fields become active so that you can set the range of the scan.
3. Adjust the Step Width as necessary. The default setting of 0.1 μm is suitable for most purposes. A shorter step width requires a longer scan time, while a larger step width reduces the resolution of the measurement.

 

NOTE:
A shorter Step Width requires more measurements. Hardware limitations only allow 40000 steps, limiting either the scan range or the step width.

 

4. Adjust the Dwell Time as necessary. This needs to be in the range 14…1092 μs. The default is 100 μs.
5. Check the Calc Width box. The width will be calculated after the measurement. You can also check this button after the measurement to perform the calculation.
6. Click Scan to start the anode scan. This will take a couple of minutes. When completed, a line is fitted to the edge and the edge width is calculated.
