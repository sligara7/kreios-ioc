Operating the Source

After connecting to the device, you can switch it on and change operating parameters.

 

As with the controls on the front panel of the UXC 1000, the recommended way to operate is by using presets:

1. Select a preset from the Preset menu.
2. Click Activate Preset. The source will start with the parameters in the preset.

 

There are two other buttons at the bottom of the Device Control that are used for operation:

■    Standby—switches off the high voltage and sets the emitter to the standby current.

■    Off—switches off the high voltage and the emitter current. The water cooling is also switched off.

 

 

You can manually change the operating settings. Note the following points:

■    You need to activate a parameter by checking the box next to it before you can change its value.

■    Limits for the parameters are also set in the device control. See Additional Controls for more details.

■    If the control mode is set to Independent, the Power and Spot Size settings are restricted to "safe" values in order to prevent damage to the source. Changing one of these values may lead to the other being automatically adjusted in order to protect the anode.

■    When using the Power control mode, Spot Size cannot be manually changed and it is grayed out.

■    When using the Spot Size control mode, Power cannot be manually changed and it is grayed out.

 

After setting the operating parameters, you can store them in a preset.
