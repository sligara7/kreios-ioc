XR 50 Device Control

The XR 50 Device Control allows you to control an X-ray source. You can set all operating parameters and save parameter sets as templates for future applications, using the standard SpecsLab Prodigy functionality.

 

The following topics are covered:

■    Switching on the X-ray source and setting its operating status.

■    Adjusting the operating parameters.

■    An overview of all features in the Device Control.

 

 

Before using the XR 50 Device Control, you need to make a connection:

■    Click Connect. There will be a pause while the connection is made. On establishing the connection, the Device Control frame turns blue. If you have configured the device as part of an experiment in the Experiment Editor, the Device Control is already connected.

 

When designing an electron spectroscopy experiment, you need to add an excitation source. The XR 50 can be added to the experiment—it will be switched on and off automatically as part of the experiment schedule. The source controls in the Experiment Editor allow you to select the anode and setting the operating parameters.

 

Clicking the Disconnect button breaks contact with the instrument. All controls are inactive. There are two buttons, and , which collapse and expand the Device Control respectively.
