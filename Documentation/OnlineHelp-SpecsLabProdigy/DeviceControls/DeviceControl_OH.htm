Device Controls

The Device Controls view contains control panels for all SPECS devices on your system. The view is divides into two sections:

■    The Devices pane shows all available devices. Connected devices are indicated by a tick. You can toggle the connection status of a device by clicking its option box.

■    The main pane in the view contains the available devices. These offer controls for setting the device parameters. You can hide inactive devices by clicking the icon in the toolbar. You can rearrange the device controls for a better overview.

 

The Device Control view also has a toolbar to make global changes to the devices in the view.

 

NOTE:
If the Device Control View does not contain an instrument on your system, you need to run the SpecsLab Prodigy installer, selecting the appropriate device, in order to reconfigure the program. Please contact SPECS support if necessary.
