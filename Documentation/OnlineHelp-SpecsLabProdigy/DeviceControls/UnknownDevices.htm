Unknown Devices

Unknown devices are indicated by a question mark icon. This is shown when a file contains details of a device, but the device is not configured in SpecsLab Prodigy. This may occur for the following reasons:

■    When data acquired on one installation of SpecsLab Prodigy is opened in another.

■    If a device is reconfigured. The configuration in the data file is then different from the current configuration in SpecsLab Prodigy.

 

Although it is not configured, the data file contains metadata which describes the device. This metadata is shown in SpecsLab Prodigy with the settings used in the experiment. In most cases, this should be sufficient for you to see what equipment was used and the operating parameters.
