Setting the IP Address

The IP address and port number of each device is preset at the factory and the setting is configured for use.

 

Should you need to change the communication settings of a device you can perform this in the SpecsLab Prodigy Configuration Tool. Please see the SpecsLab Prodigy Quick Guide for more details. Also check the manual of the device for information about changing the IP address of the device as well as procedures for changing the software configuration.
