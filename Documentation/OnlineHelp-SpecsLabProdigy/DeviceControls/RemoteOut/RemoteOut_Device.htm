Remote Out Devices

A remote out device allows you to run scripts or control devices that are not directly supported by SPECS. It uses a text-based protocol to send device commands over TCP, UDP or a serial connection. Any instrument which can be controlled in this way can be adapted for use in SpecsLab Prodigy.

 

For more information about configuring and using remote out devices, please see the PDF manual "Remote Out Devices" in the Documentation directory of the SpecsLab Prodigy installation.

 

Since they are user-defined for a specific instrument, the appearance, usage and even name of remote out devices are dependent on the configuration. The screenshot below shows the appearance of an example remote out device used for controlling a custom manipulator.

 

On expanding the device, there may be tabs. Each tab (e.g. Get Positions in the picture below) represents a command that can be executed by the user. The commands in these tabs will also be present in the Experiment Editor and can be added to an experiment.
