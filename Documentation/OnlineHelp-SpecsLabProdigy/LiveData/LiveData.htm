Live Data View

The Live Data View displays the variation of selected parameters over time. You can save the data for later examination in the Data History View. Most operating parameters of installed equipment are available, although some of these parameters are not of interest for data recording.

 

NOTE:
Detector and analyzer parameters (e.g. data signal) cannot be observed using the Live Data View. The Live View and data acquisition procedures are intended for use with detectors and analyzers.

 

You can adapt the selection of parameters and according to your experimental requirements, e.g. monitoring the pressure during bakeout, producing a record of automated processes or recording equipment parameters during measurements.

 

To open the Live Data View:

■    Select Views/ Live Data View from the toolbar. The view will open at the bottom of the main window. You can move this to a new position as desired.

 

The following procedures and features are available:

■    Selecting parameters.

■    Viewing options.

■    Saving data.

■    Clearing data.
