Managing Dataset Files

SpecsLab Prodigy stores calibration data in text files known as dataset files. There is a text file for each dataset file. After installation, the dataset files are stored in the database\datasetCalib1d folder in your SpecsLab2 installation directory. Calibration files have a .calib1d extension. You can open these files in a text editor to view their contents; however, you should not change their contents.
