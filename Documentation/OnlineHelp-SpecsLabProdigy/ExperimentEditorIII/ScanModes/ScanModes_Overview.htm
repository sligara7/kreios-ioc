Scan Modes

SPECS analyzers can operate in a number of different modes. You set the mode for each spectrum during definition in the Experiment Editor. Each mode has its own characteristics; available parameters and limits also vary between the modes.

 

The table below summarizes the available scan modes, together with a brief description of their major uses. The following subsections describe the modes in more details.

 

Scan mode Description
Fixed analyzer transmission Normal scanning mode for XPS, UPS, ISS.
Fixed retarding ratio Normal scanning mode for AES.
Fixed energy Measures single energy, ideal for analyzer set up and stability checks.
Snapshot Uses energy resolution of multi-channel detectors for fast peak measurements.
Detector voltage scan For finding optimum detector operating voltage.
Constant final state Scans photon energy (synchrotron) and measures fixed kinetic energy. For band structure mapping experiments.
Constant initial state Scans photon energy (synchrotron) and scans kinetic energy. For band structure mapping experiments.
