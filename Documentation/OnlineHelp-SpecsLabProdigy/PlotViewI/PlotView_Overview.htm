Plot View I: Features

The Plot View contains a chart that displays data from selected regions. You can choose which data is shown in the Data window; you can also create additional Data windows in order to compare spectra.

 

The Plot View is divided into three regions

■    Toolbar

■    Data browser

■    Chart area
