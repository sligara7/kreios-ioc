Data Browser

The data browser displays all spectra from the experiment as well as data operations performed on the spectra. You can change the appearance of the line in the Plot View and hide spectra.

By clicking the icon, you can see more information about the spectrum, including which spectrum components are displayed and additional data display settings.

 

NOTE:
The columns in the table (e.g. Scan in the screenshot below) depend on what data is available in the scan. For a single-scan, single-channel data set, for example, no column would be shown.
