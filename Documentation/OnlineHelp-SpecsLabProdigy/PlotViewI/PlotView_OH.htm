Plot View

The Plot View displays data acquired by SpecsLab Prodigy. During acquisition, the Plot View is updated so you can see the progress of your experiment.

 

To open the Plot View:

■    Select Views/ Plot View from the menu bar.

 

The Plot View is divided roughly into two regions

■    Data Browser, allowing you to select spectra and data operations, and view their properties.

■    Data Window, which displays the spectra and operations. It contains a toolbar which has a number of controls.
