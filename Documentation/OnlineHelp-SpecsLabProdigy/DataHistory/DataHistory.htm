Data History View

The Data History View is the counterpart to the Live Data View. It allows you to load and view data saved in the Live Data View. This arrangement allows you to view live data and old data logs simultaneously.

 

To open the Data History View:

■    Select Views/ Data History View from the toolbar. The view will open at the bottom of the main window. You can move this to a new position as desired.

 

You can perform the following actions in the Data History View:

■    Open data.

■    View data.

■    Save copies of the data.
